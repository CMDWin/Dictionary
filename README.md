# Dictionary
Эта программа сильно поможет Вам, если Вы быстро хотите найти перевод или значение слова, но у Вас нет интернета!
Просто откройте программу, напишите слово и мгновенно получите результат!

Не требует установки! 
Не обязательно скачивать дополнительные базы, всё необходимое уже есть в директории программы! 
Работает оффлайн! 
База содержит в себе 75000+ слов каждого языка! 
Есть возможность добавить свои слова через эту же программу!
Вы можете написать собственную базу на основе существующей!

Проверено на Windows 8.1 "Professional"
